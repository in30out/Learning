# IP
IP地址是设备在互联网中的地址（唯一身份证）
**IPv4**: 32位二进制    例：132.12.86.125
**IPv6**:128位二进制 
特别的，**127.0.0.1**为本机IP地址（**本机回环地址**）

# 端口（Port）
端口号是整数，取值范围在0-65535,它是用来标识计算机设备中的运行中的程序
**HTTP协议**默认端口号为**80**，**HTTPS协议**默认端口号为**443**
访问格式： **IP：端口**   例： 110.242.69.21：443
**本机域名**：**localhost**
